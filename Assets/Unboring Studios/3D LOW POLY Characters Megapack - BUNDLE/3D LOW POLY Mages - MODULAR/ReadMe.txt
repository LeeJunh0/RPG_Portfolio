You want more animations for these characters?

The characters are designed to work perfectly in combination with the animation assets from Publisher "Kevin Iglesias". 

Follow these steps:
- Download one of the assets from Unity Asset Store, for example:
    - Basic Motions FREE (https://assetstore.unity.com/packages/3d/animations/basic-motions-free-154271)
    - Basic Motions (https://assetstore.unity.com/packages/3d/animations/basic-motions-157744)
    - Caster Mage Animations (https://assetstore.unity.com/packages/3d/animations/caster-mage-animations-156596)
- Import the package in Package Manager
- Click on one of the characters
- Replace the Controller in the "Animator" Component
- Have fun!
